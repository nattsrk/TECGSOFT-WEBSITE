◉ Live OT
Video Sources
ButtonFeed

🔬 LaparoscopeInternal laparoscopic camera feed inside patient

📺 All MonitorsGrid view — OT Camera (large) + Laparoscope + Endoscope. Click secondary to swap to primary. Click primary to exit to single view.
Live Annotation
Click ✎ Annotate in the controls bar to open the annotation toolbar over the live video.
ToolHow to use
✎ PenClick & drag to draw freehand lines
→ ArrowClick start point, drag to endpoint — draws directional arrow
○ CircleClick centre, drag outward to set radius
Colour swatchesRed, Yellow, Cyan, White — click to switch
🗑 ClearRemoves all annotations from the canvas
📷 Save FrameTakes a snapshot of the annotated frame — appears in Snapshots panel
✕ ExitClick red ✕ in toolbar, or click the ✓ Annotating button in controls bar to exit
⚠ Annotations are not auto-saved — they live in memory only. Use the 📷 Save Frame button in the toolbar to capture before clearing or exiting.
Recording
Click 🔴 Start Recording → timer starts, recording is AES-256 encrypted. Click ⏹ Stop Recording to end. The live feed always keeps playing — stop recording never interrupts the video. (OT Staff role cannot start recordings.)
Snapshots
Click 📷 Snapshot or press S → frame captured from current source → appears in Snapshots panel → click ⬇ Save to download as JPG. Works with annotated frames too.
Audio
All feeds start muted (browser requirement). Click 🔇 Unmute to enable sound. When Voice Commands are active, video auto-mutes to prevent mic interference — restored when voice is stopped.
📂 Archive Search & Filter
The Archive page provides real-time filtering across all recordings:

Text search: Type any part of a doctor name, procedure, OT room, or date — rows filter instantly.
Doctor dropdown: Filter to a specific surgeon's cases.
Procedure dropdown: Filter by procedure type (Cholecystectomy, Arthroscopy, etc.).
Date range: Set From / To dates to see cases within a period.
Reset button: Clears all filters instantly.
Result count: Shows "X recordings" matching current filters.

Each recording row shows a 🔒 AES-256 badge confirming it is encrypted at rest. Download buttons are hidden for OT Staff role.
