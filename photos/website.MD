## DESIGN SPECIFICATION

### Color Palette
```
--bg-primary:      #0A0A1A    (Deep space navy)
--bg-secondary:    #111127    (Card backgrounds)
--bg-card:         #1A1A3E    (Section backgrounds)
--gold-primary:    #C5A355    (Primary gold accent)
--gold-light:      #D4AF37    (Hover gold)
--gold-glow:       rgba(197, 163, 85, 0.15)
--text-primary:    #FFFFFF
--text-secondary:  #A0A0B8
--text-muted:      #6B6B8D
--accent-teal:     #00A8A8    (HealthEdge)
--accent-purple:   #7C3AED    (BuildX)
--border:          rgba(197, 163, 85, 0.2)
```

### Typography
```
--font-display:    'Syne', sans-serif          (Headings)
--font-body:       'DM Sans', sans-serif        (Body)
--font-mono:       'JetBrains Mono', monospace  (Stats, tech labels)
```

### Key Animations
1. Hero headline: Letter-by-letter typewriter with gold cursor
2. Neural network background: SVG nodes with animated gold connections
3. Pillar cards: Staggered fade-up on scroll
4. Stats counters: Count-up on scroll intersection
5. AI comparison: Slide-in from left/right
6. Product cards: Horizontal drag-scroll with gold glow on hover
7. Industry cards: 3D tilt on hover
8. Navbar: Transparent → blur backdrop + gold border on scroll

### Mobile (< 768px)
- Pillars stack vertically
- Stats: 2x2 grid
- Products: Swipeable carousel
- Industries: 2 per row
- Hamburger nav with slide-in drawer
